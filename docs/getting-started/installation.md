# Installation

## Prerequisites

- Node.js 20 or later
- npm

## Install

```bash
git clone <your-repo>
cd your-project
npm install
```

## Development

Start the dev server:

```bash
npm run dev
```

## Build

Build the static site:

```bash
npm run build
```

The output will be in the `dist/` directory.
