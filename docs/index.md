# Welcome to Statix

Statix turns a directory of Markdown files into a navigable static website.

## Quick Start

1. Drop your `.md` files into the `docs/` directory
2. Run `npm run build`
3. Deploy the `dist/` folder

That's it. Your site is ready.

## Features

- **Sidebar navigation** derived from your file tree
- **Frontmatter support** for titles and ordering
- **Code syntax highlighting** out of the box
- **Light and dark mode** via OS preference
- **Mobile responsive** with hamburger menu
